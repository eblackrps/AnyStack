# Changelog
All notable changes to the AnyStack suite will be documented here.

## [1.0.0] - 2026-03-07
### Added
- Initial release of the AnyStack Enterprise Module Suite.
- **115+ Production-grade cmdlets** covering vSphere 8.0 U3 and VCF 5.x.
- **27 Specialized Modules** (Storage, Network, Security, Lifecycle, SDDC Manager, etc.).
- **Reporting Engine** with HTML export.
- **Configuration as Code** framework for declarative vSphere syncs.
- **Automated Installer** with PowerCLI dependency check.
- **CI/CD Pipeline** for GitHub Actions.
- Full support for `-WhatIf` and `-Confirm` across the entire suite.
