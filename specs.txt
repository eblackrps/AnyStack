# AnyStack Enterprise Module Suite - Technical Specifications v1.5.0
VCF.AlarmManager
- Get-AnyStackActiveAlarm
  Query TriggeredAlarmState from AlarmManager view.
  Return: Entity, AlarmName, Status, Time, AcknowledgedByUser

VCF.ApplianceManager
- Get-AnyStackVcenterDiskSpace
  Use $vi.ExtensionData.Content.Setting; parse /storage/seat, 
  /storage/core, /storage/log partitions.
  Return: Partition, UsedGB, FreeGB, TotalGB, UsedPct
- Restart-AnyStackVcenterService
  RestartService() on ServiceSystem view. -WhatIf required.
  Return: ServiceName, PreviousState, NewState, Success
- Start-AnyStackVcenterBackup
  POST to VAMI https://<vcenter>:5480/api/appliance/recovery/backup/job
  -WhatIf required.
  Return: JobId, Status, StartTime, BackupLocation
- Test-AnyStackVcenterDatabaseHealth
  Query $vi.ExtensionData.Content.HealthStatusManager.
  Return: OverallHealth, Components

VCF.AutomationOrchestrator
- Get-AnyStackFailedScheduledTask
  Query ScheduledTaskManager; filter State -eq 'error'.
  Return: TaskName, LastRun, NextRun, ErrorMessage, AffectedObject
- New-AnyStackScheduledSnapshot
  ScheduledTaskManager.CreateScheduledTask() with RecurrentTaskScheduler.
  -WhatIf required.
  Return: TaskName, ScheduleType, NextRun, TargetVm
- Set-AnyStackEventWebhook
  Configure EventHistoryCollector filter and HTTP action via EventManager.
  -WhatIf required.
  Return: WebhookUrl, EventTypes, FilterApplied
- Sync-AnyStackAutomationScripts
  Compare local script hash against vRO library; update on drift.
  -WhatIf required.
  Return: ScriptsChecked, ScriptsSynced, ScriptsSkipped, Errors

VCF.CapacityPlanner
- Export-AnyStackCapacityForecast
  Collect 30-day metric history (cpu.usage.average, mem.usage.average)
  via PerformanceManager; project 90-day linear trend; export HTML report.
  Return: ReportPath, ClustersAnalyzed, ProjectionDate
- Get-AnyStackDatastoreGrowthRate
  Query datastore FreeSpace delta over 7 days via PerformanceManager.
  Return: DatastoreName, CurrentFreeGB, GrowthRateGB_per_Day, DaysUntilFull
- Get-AnyStackZombieVm
  Get all VMs; filter PowerState = PoweredOff AND 
  LastModifiedTime > 90 days ago.
  Return: VmName, PowerState, LastModified, SizeGB, Datastore
- Set-AnyStackRightSizeRecommendation
  Analyze CPU/mem via PerformanceManager; generate recommendation.
  -WhatIf required (apply only with -Confirm).
  Return: VmName, CurrentCpu, RecommendedCpu, CurrentMemGB, RecommendedMemGB, PotentialSavings

  CertificateManager view ReplaceHostCertificate(). -WhatIf required.

- Update-AnyStackVcsCertificate
  POST to VAMI /api/vcenter/certificate-management/vcenter/tls with PEM.
  -WhatIf required.
  Return: Subject, ExpiresOn, Thumbprint, Applied

VCF.ComplianceAuditor
- Export-AnyStackAuditReport
  Run Get-AnyStackNonCompliantHost; format per-host findings with
  remediation steps; export HTML.
  Return: ReportPath, HostsAudited, NonCompliantCount, FindingCount
- Get-AnyStackNonCompliantHost
  Query HostComplianceManager; filter ComplianceStatus != 'compliant'.
  Return: Host, BaselineProfile, NonCompliantSettings, LastChecked
- Invoke-AnyStackCisStigAudit
  Check ESXi against CIS VMware ESXi 8.0 Benchmark:
  SSH state, NTP configured, lockdown mode, syslog, password policy,
  account lockout, DCUI access, ESXi shell timeout, MOB disabled.
  Return: Host, FindingsCount, Findings
- Repair-AnyStackComplianceDrift
  For each non-compliant setting from Invoke-AnyStackCisStigAudit,
  apply remediation. -WhatIf required.
  Return: Host, RemediationsApplied, RemediationsFailed, SkippedManual

VCF.DRValidator

  export HTML with per-VM status table.
  Return: ReportPath, VmsChecked, ReadyCount, NotReadyCount

  -WhatIf required.
  Return: VmName, IssuesFound, IssuesFixed, IssuesRequiringManualIntervention
- Start-AnyStackVmBackup
  CreateSnapshot_Task per VM for backup; integrate with vSphere Backup API.
  -WhatIf required.
  Return: VmName, SnapshotName, BackupJobId, Status
- Test-AnyStackDisasterRecoveryReadiness

  HA admission control enabled, cross-site network reachable via ICMP.
  Return: VmName, SnapshotAge, ReplicationRPO, HaEnabled, NetworkReachable, OverallReady
